## Steps to install this project

##### 1. Clone the repository
```
$ git clone https://github.com/breatheco-de/exercise-contact-list-context.git
```
##### 2. Install the /node_modules
```
$ npm install
```
##### 3. Run the webpack development server
```
$ npm run start
```
